Outreach

  • 2018 Girls Inc. Eureka! Summer Workshop: An Introduction to Creative Computing with Scratch

    I volunteered for the Girls Inc. Eureka! summer workshop that aims to increase the participation of women in Computer Science. It was five days workshop of two hours each session. The program was designed and run by Prof Benjamin Marlin using MIT Scratch programming. This year we had girls from Grade 7 and we had a lot of fun building games and stories using simple programming.
